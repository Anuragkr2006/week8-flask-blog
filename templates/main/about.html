About FlaskBlog

A clean, modern and responsive blogging platform developed using Flask Framework.

Project Overview

FlaskBlog is a blogging platform where users can register, login, create posts, edit their blogs, delete posts and interact using comments.

This project is developed as a Week 8 Flask Assignment using modern web development practices.

Technologies Used

  • Flask
  • SQLAlchemy
  • Flask-Login
  • Flask-WTF
  • SQLite
  • Bootstrap 5
